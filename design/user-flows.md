# TLB Diamond Marketplace User Flows

## 🎯 User Journey Overview

### Core User Types
1. **👤 Buyer** - Purchases products using TLB Diamonds with installment plans
2. **🏪 Seller** - Lists products and receives guaranteed payments
3. **🤝 Affiliate** - Refers users and earns lifetime commissions
4. **👨‍💼 Admin** - Manages platform operations and resolves disputes

```
🎧 Multi-Role Support System
├── Ticket Creation (Any User)
├── Initial Triage (Admin)
├── Specialist Assignment
│   ├── Technical Issues → Tech Admin
│   ├── Payment Issues → Finance Admin
│   ├── Seller Issues → Merchant Admin
│   └── Device Issues → Security Admin
├── Resolution Process
├── User Notification
└── Satisfaction Survey
```

### Dispute Resolution Flow
```
⚖️ Dispute Management
├── Dispute Initiation (Buyer/Seller)
├── Evidence Collection
├── Mediation Process (Admin)
├── Investigation
├── Decision & Resolution
├── Appeal Process (if applicable)
└── Final Settlement
```

### Community Events Flow
```
🎉 Community Engagement
├── Event Planning (Admin)
├── Affiliate Promotion
├── User Participation
├── Bonus Distribution
├── Community Feedback
└── Success Measurement
```

This comprehensive user flow design ensures smooth interactions across all user types while maintaining the security and compliance requirements of the MDM lock system.
